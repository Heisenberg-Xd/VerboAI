## рҹҡҖ Overview

**VerboAI** is an advanced AI-powered intelligence platform designed to analyze large collections of multilingual documents and convert them into structured knowledge and actionable insights.

The system leverages modern **Natural Language Processing** and **Machine Learning** techniques to automatically detect languages, translate documents, generate semantic embeddings, cluster topics, extract keywords, perform sentiment analysis, and organize information into a fully searchable knowledge base.

With integrated **AI Chat (RAG)** capabilities, users can interactively query their documents and receive accurate, context-grounded answers вҖ” all from a single unified platform.

## вңЁ Key Features

| Feature | Description |
|---|---|
| рҹҢҚ **Multilingual Intelligence** | Automatically detects document languages and translates them into English for unified analysis |
| рҹ“Ӯ **Document Processing Pipeline** | Upload multiple documents and run them through a full AI intelligence workflow |
| рҹ§  **Semantic Understanding** | Uses **Sentence-BERT** embeddings for deep semantic representation |
| рҹ§© **Topic Clustering** | Groups documents into coherent clusters using **KMeans** clustering |
| рҹ“қ **Automatic Summaries** | Generates concise, readable summaries for each document cluster |
| рҹ”‘ **Keyword Extraction** | Extracts top keywords from documents using **TF-IDF** |
| рҹҳҠ **Sentiment Analytics** | Analyzes document tone and sentiment using **VADER** |
| рҹ“Ҡ **Cluster Visualization** | Visualizes document clusters using **PCA** dimensionality reduction |
| рҹ—Ӯ **Knowledge Base Generation** | Automatically organizes documents into structured, topic-based folders |
| рҹ“‘ **Intelligence Reports** | Generates detailed reports containing insights, statistics, and analytics |
| рҹӨ– **AI Chat with RAG** | Ask natural language questions and receive contextual AI-grounded answers |

## рҹ“Ұ Installation

### Prerequisites

- Python **3.9+**
- pip or conda package manager
- 4GB+ RAM recommended for large document collections

### Setup

```bash
# 1. Clone the repository
git clone https://github.com/your-org/verboai.git
cd verboai

# 2. Create a virtual environment
python -m venv venv
source venv/bin/activate        # Linux / macOS
# venv\Scripts\activate         # Windows

# 3. Install dependencies
pip install -r requirements.txt

# 4. Download language models
python scripts/download_models.py
```

## вҡЎ Quick Start

```python
from verboai import VerboAI

# Initialize the platform
platform = VerboAI(output_dir="./knowledge_base")

# Upload and process documents
platform.ingest_documents("./my_documents/")

# Run the full intelligence pipeline
results = platform.run_pipeline(
    translate=True,
    cluster=True,
    sentiment=True,
    generate_report=True
)

# Start the AI chat interface
platform.chat("What are the main themes across the documents?")
```

## рҹ”§ Configuration

Create a `config.yaml` file in the project root to customize the pipeline:

```yaml
pipeline:
  embedding_model: "all-MiniLM-L6-v2"   # Sentence-BERT model
  num_clusters: 8                         # KMeans cluster count
  max_keywords: 15                        # TF-IDF keyword limit
  translation_target: "en"               # Target translation language

rag:
  model: "gpt-4o-mini"                   # LLM for chat
  top_k_documents: 5                     # Documents retrieved per query
  temperature: 0.3

output:
  save_reports: true
  report_format: "pdf"                   # pdf | html | markdown
  visualize_clusters: true
```

## рҹ“Ҡ Pipeline Stages

```
Stage 1 в”Җв”Җв–¶  Document Ingestion      Upload PDFs, DOCX, TXT, HTML
Stage 2 в”Җв”Җв–¶  Language Detection      Identify source languages
Stage 3 в”Җв”Җв–¶  Translation             Normalize to English via NMT
Stage 4 в”Җв”Җв–¶  Embedding Generation    Sentence-BERT semantic vectors
Stage 5 в”Җв”Җв–¶  Topic Clustering        KMeans group discovery
Stage 6 в”Җв”Җв–¶  Keyword Extraction      TF-IDF per cluster
Stage 7 в”Җв”Җв–¶  Sentiment Analysis      VADER tone classification
Stage 8 в”Җв”Җв–¶  Cluster Visualization   PCA 2D/3D projection
Stage 9 в”Җв”Җв–¶  Knowledge Base Build    Structured folder output
Stage 10 в”Җв–¶  Report Generation       Full analytics export
Stage 11 в”Җв–¶  RAG Chat Interface      Interactive document Q&A
```

## рҹӨ– AI Chat (RAG)

VerboAI's Retrieval-Augmented Generation chat lets you ask questions directly about your documents:

```python

response = chat.ask("What are the key risks mentioned across all reports?")
print(response.answer)
```
